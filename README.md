# google-sync

1. install gam (gapps admin)
    ```
    bash <(curl -s -S -L https://git.io/install-gam)
    ```
2. install requirements.txt
3. create .env
    ```
    GOOGLE_STUDENTS_DOMAIN
    DATAGUN_DIR
    DATAGUN_VIRTUALENV
    GAM_USERS_EXPORT_FILE
    DB_USERS_EXPORT_FILE
    GAM_ADMINS_EXPORT_FILE
    DB_ADMINS_EXPORT_FILE
    ```
